What is this class?

This class is introduction to UI design with HTML, CSS, and Javascript. You will learn how to create and host a static website on GitHub Pages.

Each participant will get to make a website: either in a group, or alone. At the end of the quarter, everyone will have a finished product they can be proud of!

This class will be most successful if everyone works in groups. The best part of Software Engineering is collaboration and exchange of ideas! I highly encourage everyone to work with others on your project.
